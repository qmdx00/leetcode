### [爬楼梯](https://leetcode-cn.com/problems/climbing-stairs/)

#### 解题

##### 解法1

```java
class Solution {
    public int climbStairs(int n) {
        int[] F = new int[n + 2];
        F[1] = 1;
        F[2] = 2;
        if (n >= 3) {
            for (int i = 3; i <= n; i++) {
                F[i] = F[i - 1] + F[i - 2];
            }
        }
        return F[n];
    }
}
```
