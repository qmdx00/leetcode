### [买股票的最佳时机II](https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock-ii/)

#### 解题

##### 解法1

> 思路：贪心算法

```java
class Solution {
    public int maxProfit(int[] prices) {
        int len = prices.length;
        int profit = 0;
        for (int i = 0; i < len - 1; i++) {
            if (prices[i] < prices[i + 1]) {
                profit += prices[i + 1] - prices[i];
            }
        }
        return profit;
    }
}
```
