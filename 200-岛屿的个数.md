### [岛屿的个数](https://leetcode-cn.com/problems/number-of-islands/)

#### 解题

##### 解法1

```java
class Solution {
    public int numIslands(char[][] grid) {
        int count = 0;
        for (int i = 0; i < grid.length; i++) {
            for (int j = 0; j < grid[0].length; j++) {
                if (grid[i][j] != '0') {
                    count ++;
                    dfs(grid, i, j);
                }
            }
        }
        return count;
    }

    static void dfs(char[][] map, int x, int y) {
        if (x < 0 || x >= map.length) return;
        if (y < 0 || y >= map[0].length) return;
        if (map[x][y] == '0') return;

        map[x][y] = '0';

        dfs(map, x, y + 1);
        dfs(map, x, y - 1);
        dfs(map, x + 1, y);
        dfs(map, x - 1, y);
    }
}
```
