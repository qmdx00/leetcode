### [两两交换链表中的节点](https://leetcode-cn.com/problems/swap-nodes-in-pairs/)

#### 解题

##### 解法1

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        ListNode res = new ListNode(-1);
        ListNode r = res;

        ListNode p, q, cur;
        cur = head;

        while (cur != null && cur.next != null) {
            p = cur;
            q = p.next;
            cur = q.next;

            q.next = p;
            p.next = null;
            r.next = q;

            r = r.next.next;
        }
        if (cur != null) {
            r.next = cur;
        }
        return res.next;
    }
}

class ListNode {
    int val;
    ListNode next;
    ListNode(int x) {
        val = x;
    }
}
```
