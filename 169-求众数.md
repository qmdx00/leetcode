### [求众数](https://leetcode-cn.com/problems/majority-element/)

#### 解题

##### 解法1

```java
class Solution {
    public int majorityElement(int[] nums) {
        int result = nums[0], count = 0;
        for(int i = 0; i < nums.length; i++) {
            if(count == 0) {
                result = nums[i];
                count++;
            } else if(result == nums[i]) {
                count++;
            } else {
                count--;
            }
        }
        return result;
    }
}
```

##### 解法2

```java
class Solution {
    public int majorityElement(int[] nums) {
        Arrays.sort(nums);
        return nums[nums.length/2];
    }
}
```
