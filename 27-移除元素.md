### [移除元素](https://leetcode-cn.com/problems/remove-element/)

#### 解题

##### 解法1

> 思路：直接遍历并覆盖元素

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        k = 0
        for i in range(len(nums)):
            nums[k] = nums[i]
            if nums[i] != val:
                k += 1
        return k
```
